Implemented character movement aligned with dynamic gravity sources.

### What changed
- character orientation aligned to gravity direction
- movement enabled on spherical surfaces
- basic support for multiple gravity attractors

### Challenges
- rotation snapping when switching attractors
- unstable transitions at high velocity

### Next steps
- smooth interpolation between gravity sources
- velocity-based rotation logic
- extended edge case testing

## March 13, 2026 — Planetary Gravity System (Chaos Physics)

**Tags:** Physics, Chaos, Gravity, Core System  
**Video:** https://youtu.be/8Zs5AkZgMYs?si=TYFQ58QVBwfV2dXV  
**Related Systems:** Physics, Movement  

![GravitySystem](../../Screenshots/Image_GravitySystem_1.png)

### Summary
Implemented a spherical gravity system using Chaos Physics callbacks.

### What changed
- added gravity attractors for spherical worlds
- applied forces to physics actors via Chaos callbacks
- refactored into reusable system

### Implementation notes
- based on Epic tutorial but redesigned for flexibility
- supports multiple attractors and dynamic interaction

### Challenges
- extremely high velocity near attractor origin
- instability close to gravity center

### Next steps
- clamp velocity near origin
- implement gravity falloff function
- stress testing with multiple attractors
